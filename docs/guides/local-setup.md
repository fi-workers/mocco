# Local development setup

> Ported from the showyourtime harness pattern. prod = `www.mocco.club` Ā· local browsing = `https://mocco.work` (traefik+mkcert). ADR 0006.

## Layout

```
Browser  ā†’ https://mocco.work  ā†’ traefik(:443, mkcert TLS) ā†’ Next(localhost:3100)
Local DB ā†’ docker compose postgres(:5432)
```

## One-time setup

```bash
make application   # brew: mkcert nss traefik node corepack
make initialize    # certs(mkcert) + hosts(/etc/hosts, sudo) + yarn install
make docker-up     # local Postgres
yarn db:generate && yarn db:migrate
```

- `make certs` ā€” `mkcert mocco.work '*.mocco.work'` ā†’ `infra/local/cert/` (gitignore)
- `make hosts` ā€” add `mocco.work`, `www.mocco.work`, `host.docker.internal` ā†’ 127.0.0.1 to `/etc/hosts`

## Development

```bash
make dev      # = yarn dev = concurrently(run-frontend + run-traefik)
              # ā†’ visit https://mocco.work (traefikā†’Next:3100)
```

## Environment variables

Copy `src/frontend/.env.example` ā†’ `.env`:
- `AUTH_URL` = `https://mocco.work`
- `DATABASE_URL` = local pg

## Login (email + password)

No external OAuth app is needed ā€” sign-up works out of the box.

1. Copy `src/frontend/.env.example` ā†’ `.env` and fill in:
   - `AUTH_SECRET` = `openssl rand -base64 32`
   - `AUTH_URL` = `https://mocco.work`
   - `DATABASE_URL` = local pg
2. `make dev` ā†’ `https://mocco.work` ā†’ **Create account** ā†’ you land on `/account`.

- Route: `app/api/auth/[...all]` re-exports the backend's neutral `authHandler` (the vendor is isolated inside `src/backend/auth/`)
- Tables: `mocco_users` / `mocco_sessions` / `mocco_accounts` / `mocco_verifications` (uuid PKs generated by the DB)
- Smoke test: `curl .../api/auth/get-session` ā†’ `null` (200) means the wiring is correct.
- Google SSO and GitHub account-linking land later as separate PRs; workspaces are created from the app after sign-in (see [workspace model](../reference/workspace.md)).

## Webhooks & OIDC callbacks (later ā€” GitHub App integration)

GitHub webhooks and Actions callbacks need a publicly reachable URL, which local browsing does not.
A cloudflared tunnel (`dev.mocco.work` ā†’ localhost:3100) will be reintroduced together with the
GitHub App integration; it is intentionally not part of the base local setup.
